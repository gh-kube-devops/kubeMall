# Step 3：配置 application.yml（10分钟）

## 🤖 AI给你模板（直接用）

```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/kubemall
    username: postgres
    password: 123456
  jpa:
    hibernate:
      ddl-auto: update
    show-sql: true
    properties:
      hibernate:
        format_sql: true

server:
  port: 8081
```

# Step 4：写 Entity（10分钟）

## 🤖 AI生成（直接用）

```java
@Entity
@Table(name = "users")
@Data
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    private String username;
    private String password;
    private String email;
}
```

---

# Step 5：写 Repository（5分钟）

```java
public interface UserRepository extends JpaRepository<User, Long> {
}
```

---

# Step 6：写 Controller（15分钟）

## 🤖 AI生成 + 🧠你理解

```java
@RestController
@RequestMapping("/users")
public class UserController {

    @Autowired
    private UserRepository userRepository;

    @GetMapping
    public List<User> getAllUsers() {
        return userRepository.findAll();
    }
}
```

# Step 9（加分）：自己扩展一个接口（20分钟）

## 🧠 必做（关键）

自己写一个：

👉 根据ID查询用户

```java
@GetMapping("/{id}")
public User getUserById(@PathVariable Long id) {
    return userRepository.findById(id).orElse(null);
}
```

访问：

# 🚨 常见问题（提前避坑）

### 1. 启动报错：连接数据库失败

👉 检查：

* PostgreSQL 是否启动
* 用户名密码
* 端口 5432

---

### 2. 表不存在

👉 执行 SQL 或检查数据库名称

---

### 3. 端口被占用

👉 改：

```yaml
server:
  port: 8082
```
